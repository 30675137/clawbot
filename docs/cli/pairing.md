# `clawbot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
clawbot pairing list whatsapp
clawbot pairing approve whatsapp <code> --notify
```
