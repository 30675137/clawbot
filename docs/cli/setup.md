# `clawbot setup`

Initialize `~/.clawdbot/clawbot.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
clawbot setup
clawbot setup --workspace ~/clawd
```

To run the wizard via setup:

```bash
clawbot setup --wizard
```
